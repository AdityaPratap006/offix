React Native Support

Offix Client can be used within React Native by providing some React Native specific interfaces that override the original usage of browser APIs.

Network Status

Firstly, a React Native NetworkStatus implementation should be defined in your application.

import {  NetInfo } from "react-native"
import { NetworkStatus, NetworkStatusChangeCallback } from "apollo-offline-client"

class ReactNativeNetworkStatus implements NetworkStatus {
  public onStatusChangeListener(callback: NetworkStatusChangeCallback): void {
    const listener = (connected: boolean) => {
      callback.onStatusChange({online: connected})
    };
    NetInfo.isConnected.addEventListener('connectionChange', listener)
  }

  public async isOffline(): Promise<boolean> {
    const isConnected = await NetInfo.isConnected.fetch()
    return !isConnected
  }
}

This implementation uses the React Native NetInfo API to determine the network state.

Storage

In React Native AsyncStorage is typically used. You can use either AsyncStorage implementation:

import AsyncStorage from '@react-native-community/async-storage';

or

import { AsyncStorage } from 'react-native';

Then define the storage interface:

const storage = {
  getItem: async (key) => {                                  
    const data = await AsyncStorage.getItem(key);
    // required to prevent data becoming corrupt and bloated
    if (typeof data === 'string') {
      return JSON.parse(data);
    }
    return data;
  },
  setItem: async (key, value) => {
    let valueStr = value;
    if (typeof valueStr === 'object') {
      valueStr = JSON.stringify(value);
    }
    return AsyncStorage.setItem(key, valueStr);
  },
}
