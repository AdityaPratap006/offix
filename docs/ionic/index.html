Ionic and Apache Cordova integration

Offix supports the Cordova platform out of the box by utilizing existing Apache cordova plugins. To use Offix with Apache Cordova (Ionic) please make sure that your project has Network Information Plugin installed.

More information: https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-network-information/

Example Ionic Application#

Offix is implemented in Angular based Ionic example application. For more information please check:
